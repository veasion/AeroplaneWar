# [AeroplaneWar 飞机大战](https://github.com/veasion/AeroplaneWar "飞机大战")

## 游戏简介

### 这是一个用JAVA写的卡哇伊飞机大战小游戏，积分制，包含随机敌方飞机+Boss，随机背景及音乐，有武器补给，血量补给，其中武器补给非常丰富。
